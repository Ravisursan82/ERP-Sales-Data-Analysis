# ERP-Sales-Data-Analysis  

This project analyzes sales data to identify patterns and trends using Python, and Tableau.  

Tools Used  

Python for initial data cleaning,exploration,data processing and analysis  

Tableau for creating interactive dashboards and visualizations  

Problem Statement:  

The company sells ERP products across multiple Indian cities through different sales channels and license types. Management wants to understand what drives revenue performance, customer retention, and churn risk in order to improve sales effectiveness and long-term growth. Using the provided dataset, analyze sales performance, renewal behavior, and churn risk across products, license types, sales channels, cities, and sales representatives, and provide actionable insights to increase revenue and reduce churn.
